事件循环

在python和js中都有事件循环

这证明每次事件循环宏任务只取一个执行，微任务一次全部执行

setTimeout(() => {
  console.log('a');
  Promise.resolve().then(res => {
    console.log('c');
  });
});
setTimeout(() => {
  console.log('b');
  Promise.resolve().then(res => {
    console.log('d');
  });
});

解释器解析完了中间码，先执行一遍中间码，然后把各种异步任务放到各自异步任务中， 编译性语言和解释性语言是不同的两种方式

  • 编译性语言类似go，rust，是为服务器设计的，要支持并发性，
  • 而解释性语言一般都是单线程执行一些简单任务设计的，在语言层面实现一些异步任务， 事件循环就成了这些语言的选择
